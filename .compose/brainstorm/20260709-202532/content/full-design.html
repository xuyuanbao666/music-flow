定位：面向音乐爱好者的跨平台高音质播放器

核心价值：三端无缝体验 + 多源音乐接入 + 高保真播放

🎵 核心功能模块

播放器核心
  • 本地文件播放（FLAC/WAV/MP3）
  • 在线音乐流播放
  • 10 段均衡器预设
  • 播放队列管理
  • 歌词显示（LRC 格式）
  • 音频可视化（频谱图）
数据同步
  • 播放列表云同步
  • 播放进度同步
  • 收藏/喜欢同步
  • 均衡器设置同步
  • 播放历史同步
  • 本地文件 P2P 同步

🛠️ 技术栈详情

技术选型
前端框架 React 18 + TypeScript
状态管理 Zustand + TanStack Query
桌面端 Tauri v2 + Rust
移动端 Capacitor 6
音频引擎 Web Audio API + Howler.js
UI 组件 Radix UI + Tailwind CSS
构建工具 Vite + Turborepo
后端服务 Node.js + Hono + PostgreSQL
同步引擎 CRDT (Yjs) + WebSocket

  • 歌词显示
